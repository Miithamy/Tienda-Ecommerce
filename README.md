Este proyecto corresponde a la proyecto del modulo 2 , en donde se creo un ecommerce básico utilizando: Html, Boostramp, JavaScript, manipulación del DOM y almacenamiento con LocalStorage.
El objetivo es permitir agregar productos, actualizar cantidades, mostrar subtotales y total final en la página de carrito.

Tecnologías Utilizadas:
- HTML5
- CSS3
- Boostramp5
- JavaScript
- Git y GitHub para control de versiones

¿Qué permite hacer?
Agregar productos al carrito
Guardar datos en LocalStorage
Calcular subtotal (precio × cantidad)
Calcular total del carrito
Vaciar el carrito con un botón
Mensajes visuales de producto agregado
Navegar por las diferentes seccione
Visualizar descripción de cada producto
